# qmove

## Usage

```text
usage: qmove --source=SOURCE --destination=DESTINATION --category=CATEGORY [<flags>]


Flags:
      --[no-]help                Show context-sensitive help (also try --help-long and --help-man).
  -s, --source=SOURCE            Source download path
  -d, --destination=DESTINATION  Destination path
  -c, --category=CATEGORY        Category
  -p, --perm=PERM                Permission in number format
      --[no-]version             Show application version.
```
